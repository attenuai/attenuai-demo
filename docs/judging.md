# Judging Rubric

## Default

### Impact (25%)

What is the project’s long-term potential for success, growth and impact? Does it fit into one of the problem statements listed above? Is the project useful, and for who?

### Codex App (25%)

Built with Codex app – Tell a creative, end-to-end story of how Codex enabled the build with parallel agents and worktrees

### Creative use of skills (25%)

Creative use of skills – Strong practical use of skills in Codex for coding workflows, developer tasks, and “non-technical technical work”

### Demo & Pitch (10%)

Is the concept of this project novel? Is it presented and demo’d well? Is this a cool project to watch? Is this a demo, and NOT just a presentation, figma file, or anything else? How well has the team implemented the idea? How technical is their usage and implementation? Is this a project with sound engineering?

### Feasibility & Scalability (15%)

Could this realistically be implemented or expanded?
